LiftQuest

> **Work in progress** — features are actively being developed and things may change.

**Gamified workout tracker with XP system, levels, and achievements**

Track your strength training and cardio workouts while earning XP, leveling up, and unlocking achievements. Built with modern web technologies and with a help of Claude.

## Tech Stack

**Frontend:**
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Recharts (data visualization)
- Lucide React (icons)

**Backend:**
- NestJS
- PostgreSQL
- Prisma ORM
- JWT Authentication
- bcrypt

**Deployment:**
- Railway (full-stack hosting)
- Docker
